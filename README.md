# Banner-Grabber
A Simple tool to display the Service Banner.

# Features
- Easy to Use.
- Helpful comments and documentation in the generated code.
- Supported OS Linux.

# Usage
```console
git clone https://github.com/SubhankarX01/Banner-Grabber/
```
```console
cd Banner-Grabber
```
```console
python3 banner.py -h
```
OR
```console
chmod +x banner.py 
```
```console
./banner.py -h
```
